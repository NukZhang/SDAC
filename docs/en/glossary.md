# Glossary

- **SDAC**: Spec-Driven AICoding (governance-first AI coding methodology)
- **Me2AI**: Human-authored spec (requirements/constraints/iteration plan)
- **AI2AI**: AI-maintained execution state (facts, not logs)
- **Governance Layer**: SKILL + Sub Skills + control mechanisms
- **Lane / Work Lane**: Isolated work stream with allowed paths
- **Merge Gate**: Serial integration rule set controlled by Release Owner
- **Minimal Fix Diff**: Smallest auditable patch to correct violations
