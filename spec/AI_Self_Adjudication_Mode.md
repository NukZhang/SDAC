# AI 自我裁决模式（违规 → 自动回滚）

> 当 Checklist 命中 A/B 级违规时必须启用。

## 输入
- 最近一次输出（代码/方案/文件变更）
- 命中的违规项（A/B/C）

## 规则

### A 级：全部回滚
- 放弃最近一次输出的所有变更与结论
- 回到“进入本次迭代之前”的稳定状态
- 重新执行：读 Me2AI → 对齐 AI2AI → 最小可完成集

### B 级：回退到最小可完成集
- 保留与当前迭代目标直接相关且合规的最小改动
- 删除/回退所有越界内容（包括预留抽象、跨模块重构）
- 重新补齐证据链（测试命令、验证结果、AI2AI 更新）

### C 级：不回滚，进入最小修复
- 直接进入 `spec/Minimal_Fix_Diff_Mode.md`

## 输出要求（结构化）
- adjudication: "A" | "B" | "C"
- rollbackPlan: 删除/回退哪些内容（按文件/模块列出）
- nextSteps: 下一步如何重新推进（最小集）
